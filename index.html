Web + API

支持 Docker 部署

Monorepo 架构，包含共享下载核心、Fastify API 服务器和 TanStack Start Web 客户端，一行命令即可启动。

核心 packages/downloader-core — 共享 yt-dlp/ffmpeg 下载核心
API apps/api — Fastify + oRPC + SSE 事件
Web apps/web — TanStack Start + oRPC 客户端
docker-compose.yml 
services:
  api:
    image: ghcr.io/nexmoe/vidbee-api:latest
    ports:
      - "3100:3100"
    volumes:
      - vidbee-downloads:/data/downloads
      - vidbee-data:/data/vidbee

  web:
    image: ghcr.io/nexmoe/vidbee-web:latest
    depends_on:
      - api
    ports:
      - "3000:3000"

启动服务

docker compose up -d

本地开发

pnpm run start:web
